# Rules Sources

Citaat-extracts en notities per geconsulteerde bron voor `RULES.md`. Eén bestand per bron:

- `russ.md` — Jonathan Russ, *Mancala Games* (1984) (te bevestigen, zie Open Question #3)
- `de_voogt.md` — Alex de Voogt, etnografisch werk
- `bao_society.md` — Bao Society / FIDE Mind Sports toernooiregels
- `geziefer_bga.md` — observaties uit `geziefer/baolakiswahili` als gedragsoracle

Elke bron-file documenteert relevante uitspraken met paginanummers/citaten en flags conflicten met andere bronnen. `RULES.md` synthetiseert deze tot één canonieke regelset.
